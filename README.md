# ricebox

an opinionated i3 rice with polybar, kitty, rofi, and picom. features a unified theme system that swaps colors across all configs with a single click.

## what you get

- **i3** - gaps, scratchpad, border theming, float rules
- **polybar** - heavily customized bar with clickable modules and rofi detail panels
- **kitty** - full 16-color terminal theme
- **rofi** - themed app launcher with custom panels for system info
- **picom** - glx compositor with vsync
- **12 themes** - switch instantly from the bar

## polybar modules

| module | info | click |
|--------|------|-------|
| weather | location-aware temp + icon | 3-day forecast panel, search other cities |
| slack | unread notification indicator | focus slack window |
| tailscale | exit node status | rofi menu to switch exit nodes, connect/disconnect |
| pulseaudio | volume with ramp icons | - |
| bluetooth | device connection status | left: blueman, right: toggle |
| wifi | signal strength + essid | left: nm-connection-editor, right: toggle |
| cpu | usage % | rofi panel: per-core freq, governor, top procs |
| cpu-freq | avg clock speed (GHz) | rofi panel (same as cpu) |
| memory | ram usage % | rofi panel: ram/swap breakdown, top procs |
| temperature | cpu die temp (k10temp/coretemp) | rofi panel: all hwmon sensors |
| battery | charge level with ramp icons | - |
| battery-wattage | charge/discharge watts + color-coded sparkline | - |
| power-profile | cycle saver/balanced/performance | left: cycle, right: xfce power settings |
| date | clock | - |
| powermenu | lock/reboot/poweroff | expand menu |
| theme | palette icon | rofi theme switcher |

## themes

`dark-green` `dracula` `catppuccin-mocha` `gruvbox-dark` `nord` `solarized-dark` `tokyo-night` `rose-pine` `kanagawa` `everforest` `one-dark` `cyberpunk`

click the palette icon on the bar or run `~/.config/polybar/scripts/theme-switch.sh` to switch. themes apply to polybar, kitty, i3, and rofi simultaneously.

adding a theme is just creating a new `.sh` file in `themes/` following the existing format.

## install

```bash
git clone https://github.com/f0rkz/ricebox.git
cd ricebox
./install.sh
```

the installer detects your OS (`/etc/os-release`) and picks the right package manager. supported: debian/ubuntu (apt) and arch/cachyos (pacman).

the installer will:

1. install dependencies via apt or pacman (polybar, i3, kitty, rofi, picom, etc)
2. install 0xProto Nerd Font + Font Awesome 6
3. backup existing configs and copy ricebox configs
4. generate `~/.config/ricebox.env` with auto-detected settings
5. optionally set up sudoers for power profile switching + tailscale
6. optionally install `tzupdate` for automatic timezone switching (via AUR/pipx on arch, pip on debian)

## configuration

all personal settings live in `~/.config/ricebox.env`:

```bash
RICEBOX_HOME_SSID="your-wifi-ssid"
RICEBOX_HOME_LOCATION="Your+City+State+US"
RICEBOX_DPI=192
RICEBOX_VERTICAL_MONITOR="DP-6"
RICEBOX_DEFAULT_THEME="dark-green"
RICEBOX_TEMP_SENSOR="/sys/class/hwmon/hwmon8/temp1_input"
RICEBOX_FONT="0xProto Nerd Font"
RICEBOX_FONT_SIZE=12
```

this file is gitignored and generated by the installer. edit it and re-run `install.sh` to apply.

**note**: the font must be a [nerd font](https://www.nerdfonts.com/) variant for icons to render correctly.

## dependencies

**required**: i3, polybar, kitty, rofi, picom, jq, xdotool, xclip, maim, nitrogen, curl, NetworkManager, bluez

**optional**: tailscale, xscreensaver, xfce4-power-manager, tzupdate

## multi-monitor

polybar automatically detects connected monitors. set `RICEBOX_VERTICAL_MONITOR` in your env to get a slim bar on vertical displays. the laptop panel (eDP) is skipped when external monitors are connected.

## sudoers

power profile cycling and tailscale switching need passwordless sudo for:

- `/usr/bin/tee /sys/firmware/acpi/platform_profile`
- `/usr/bin/tee /sys/devices/system/cpu/cpu*/cpufreq/energy_performance_preference`
- `/usr/bin/tailscale set --exit-node=*`
- `/usr/bin/tailscale up`
- `/usr/bin/tailscale down`

the installer handles this if you opt in.
